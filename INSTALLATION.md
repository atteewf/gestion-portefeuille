# 🚀 Installation et lancement - DossierFlow (Vite)

## ✅ Avantages de cette version Vite

- ⚡ **Démarrage ultra-rapide** (< 1 seconde)
- 🎯 **Zéro deprecated warnings**
- 🔒 **Moins de vulnérabilités** (seulement ~10 dépendances)
- 🆕 **Stack moderne 2026**
- 🔥 **Hot Module Replacement instantané**

## 📦 Installation

```bash
# 1. Dézippe le projet

# 2. Entre dans le dossier
cd dossier-flow-vite

# 3. Installe les dépendances (RAPIDE !)
npm install

# 4. Lance l'app en mode dev
npm run dev
```

## 🌐 Ouverture

L'app se lance sur : **http://localhost:5173**

(Vite utilise le port 5173 par défaut, pas 3000 comme CRA)

## 📝 Commandes disponibles

```bash
npm run dev      # Lance le serveur de dev
npm run build    # Build pour la production
npm run preview  # Prévisualise le build
npm run lint     # Vérifie le code avec ESLint
```

## 🎯 Différences avec CRA

| Create React App | Vite |
|-----------------|------|
| `npm start` | `npm run dev` |
| Port 3000 | Port 5173 |
| ~1400 dépendances | ~10 dépendances |
| Démarrage ~30 sec | Démarrage ~1 sec |
| `src/index.js` | `src/main.jsx` |
